HUD Generator - Lost Relics

This system will generate a web page that will make a request to the Lost Relics web service, retrieving your data in real time

The only difference between the two options is that, in OBS, the background is transparent, while in the browser the background is gray


Click on 'GENERATE PAGE' to download the files

After generating the files above, follow the steps below to ensure the HUD works as expected. The next steps involve installing a program that allows the creation of a web server so the page functions properly. It's important to note that this feature only works for members.
1- First, you need to enable this option within Lost Relics:

2 - It is necessary to install a program for the page to work, as it will generate a web server:
Website: Laragon
Download link: Download Laragon - Full (173 MB)

After installing and opening Laragon, follow the steps below


Start the service by clicking the 'Start All' button:


After clicking 'ROOT', a folder will open where you need to extract the files downloaded by the generator
Directory after extraction:

After extracting the files to the mentioned folder, if you followed all the steps correctly, you will be able to see your data when you access the link below. Please remember that the game Lost Relics needs to be open and the 'Query API' option must be enabled.
